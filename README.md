# COMP24011 / COMP26020: Programming Languages and Paradigms

This repository serves as a centralized portfolio compiling advanced, multi-paradigm software engineering projects engineered across the COMP26020 curriculum. The codebase explores low-level imperative resource control in C, object-oriented modernization and memory safety in C++20, declarative structural induction in Haskell, compiler optimization heuristics, and decentralized smart contract state mechanics in Solidity.
