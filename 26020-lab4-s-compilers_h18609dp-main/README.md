# COMP26020 - Lab 4

## How to Compile and Run
This is a Python script. No compilation is needed. Run the program with:
``` python3 ColourGraph.py <input_file> <output_file> ```

- `<input_file>`: Path to the input file (e.g., `input1.txt`) - must exist in the folder.
- `<output_file>`: Path to the output file (e.g., `output1.txt`) - will be created.
